                                    Credit_risk_management
This repository has a model which is best selected model to predict if a credit card user will default in next month or not.


The dataset link is here: https://github.com/Parijat1072005/credit_risk_management/releases/download/v1.0/Datasets_final.zip


Follow the below steps to check it out.


1) Make a folder named github_project.


2) Go and open it with code.


3) Open the vs code terminal and follow the steps/codes below

>>git clone https://github.com/Parijat1072005/credit_risk_management


>>cd credit_risk_management


>>python -m venv venv


>>.\venv\Scripts\activate


>>pip install -r requirements.txt


The git attributes file has nbstripout so if its not installed in your laptop or virtual environment then go install it since it will help you not gettuing distracted with git unwanted commits.


The dataset has been provided here, "https://github.com/Parijat1072005/credit_risk_management/releases/download/v1.0/Datasets_final.zip", save this dataset in the data/ folder inside the working directory.


Put the path in the files, wherever used, so that the dataframe can be created.


Now run all the files, so that output files can be saved which will be used during the website development.


Now in the terminal, runn the foillowing code:


>>python app.py
